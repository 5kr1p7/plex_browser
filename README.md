# Plex Browser

## Search devices
* [Shodan](https://www.shodan.io/search?query=%22CherryPy%2F5.1.0%22+%22%2Fhome%22)

## Preparing
`pip3 install -r requirements.txt`

## Usage
Get sections (libraries) from device:
`./plex_browser.py <IP>`

Get videos from section:
`./plex_browser.py <IP> <SECTION_ID>`
